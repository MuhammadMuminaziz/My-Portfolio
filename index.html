Tentang Saya

Halo... Saya Muhammad Mu'min Azis mahasiswa dari STMIK IKMI CIREBON, disamping perkuliahan saya telah bekerja sebagai freelancer di bidang web development selama 1+ tahun. Ini memberi saya keahlian dalam mengembangkan situs web menggunakan bahasa seperti HTML, CSS, dan JavaScript. Sebagai freelancer, saya telah bekerja dengan berbagai klien dari berbagai industri, memahami kebutuhan mereka, dan memberikan solusi kreatif. Keahlian saya mencakup bagian depan (frontend) maupun bagian belakang (backend) web dan memungkinkan penggunaan kerangka kerja codeigniter maupun laravel. Dalam beberapa tahun ke depan, saya dapat terus mengasah keterampilan saya dan memanfaatkannya untuk berbagai proyek yang menantang.
